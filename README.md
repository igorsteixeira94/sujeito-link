## :computer: Como Usar

*Necessário criar env.js com TOKEN para utilização da API do BitLy*

```shell
git clone git@github.com:igorsteixeira94/sujeito-link.git
cd sujeito-link

yarn install

# Iniciando no Android
yarn start --android

# Iniciando no IOS
yarn start --ios

```
